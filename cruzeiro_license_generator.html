Cruzeiro — Gerador de Licenças

Uso exclusivo do administrador. Não compartilhe esta página.

Cada licença é vinculada ao email do usuário. O app valida o par email + código.
Uma mesma licença funciona em quantos computadores o usuário tiver (sem limite de dispositivos).
Código gerado para
Envie o email e o código acima para o usuário. Ele insere em Configurações → Licença → Ativar licença.

Histórico

Nenhuma licença gerada ainda.
Segurança: os códigos são derivados do email via HMAC-SHA256 com chave privada. Qualquer email sempre gera o mesmo código — não é necessário banco de dados. A chave privada está em src/main.js (constante LICENSE_SECRET).

Limitação conhecida: quem descompilar o app pode extrair a LICENSE_SECRET e gerar códigos para qualquer email. Para o seu uso atual (controle pessoal), isso é aceitável.
