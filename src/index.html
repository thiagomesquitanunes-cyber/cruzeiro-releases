🔐 Senha de acesso

Defina uma senha para proteger o Cruzeiro. Ela será solicitada toda vez que o app for aberto.
🔐 Como funciona a segurança:
Ao definir uma senha, o banco de dados é criptografado com AES-256-GCM. Um código de recuperação é enviado ao seu email — sem ele, não é possível redefinir a senha. Um backup local de emergência também é salvo automaticamente.

📤 Exportar dados

JSON completo — exporta todos os dados (contas, lançamentos, patrimônio, investimentos, metas, orçamentos) em formato estruturado. Ideal para backup.
CSV — lançamentos em formato tabular, compatível com Excel e Google Sheets.
QIF — formato padrão da indústria (Quicken Interchange Format), compatível com Quicken, Microsoft Money, GnuCash e a maioria dos apps de finanças pessoais.

Backups automáticos

Um backup é criado automaticamente toda vez que o app é aberto. Os últimos 30 backups são mantidos. Para restaurar, clique em "Restaurar" ao lado do backup desejado.
